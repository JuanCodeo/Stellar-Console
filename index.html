Acerca del proyecto

Actividad académica · UCompensar · Ingeniería de Software

Stellar Console es la actividad de profundización del curso Desarrollo de Software Web Front-End. Se reorientó el dominio del portafolio original a un sistema real de consulta científica conectado a la API pública de la NASA, manteniendo la arquitectura Sass 7-1 con principios SOLID y la integración de JavaScript modular vanilla.

Toda la lógica está separada por capas: config, core, services, views y utils. Cada servicio tiene una sola responsabilidad y depende de un cliente HTTP abstracto (core/apiClient.js) que centraliza errores, caché y rate-limit.

  • APOD: /planetary/apod — imagen del día con archivo histórico.
  • DONKI: /DONKI/{FLR, CME, GST, notifications} — clima espacial.
  • NeoWs: /neo/rest/v1/feed — feed semanal de asteroides cercanos.
  • NASA Image Library: images-api.nasa.gov/search — archivo multimedia.
  • Caché: respuestas guardadas en localStorage con TTL.
  • Errores: manejo centralizado con estados de UI dedicados.
